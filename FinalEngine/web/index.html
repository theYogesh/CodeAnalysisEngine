About

‘Analysis Engine’ is an open source and platform independent engine that consists of several command line tools and an online platform to analyse and test the C/C++ programs for better architecture, efficient memory management and robustness.
The aim of this engine is to automate the process of testing and analysing the software because in today's industry programmer's time is as important as program's time.

Features:-


Performance Analyser

The Performance Analyzer records and displays execution times for functions or program by executing programs on a real processor. To be more precise it executes the programs many times and displays the average time. You may use the information gathered by the performance analyzer to determine which functions are consuming the most time, you can evaluate them for possible performance improvements.


Code Formatter

Despite all improvements done in computing and AI, human logic analysis still cannot be fully replaced by automation. Programmers analyse the logic of the code by reading it. Code Formatter formats the program code to make it more readable and therefore reduces the chances of human error. It is necessary when a lot of programmers are working together and using each other’s code to develop an integrated module. Different programmers may use different formatting approach in their codes that makes difficult to read the final integrated code. Therefore a common formatting approach is required to make their source code more readable.


Black Box Tester

Black Box Tester tool performs black box testing on C/C++ programs without having any knowledge of the interior workings of the application. The tester will interact with the system's user interface by providing inputs and examining outputs. It can be applied to software “units” (e.g., classes, functions) or to entire programs.
